## Description

This repository contains Bash/Shell scripts and training exercises intended for 1337 students. The goal is to provide reusable resources, examples, and practice challenges to improve your Shell skills.

## Installation

```bash
# Clone the repository
git clone https://github.com/simo-SM/shellpool1337.git
cd shellpool1337
```

## Usage

* All Bash scripts are located in the `scripts/` directory.
* Read the comments inside each script to understand its purpose.

Example of running a script:

```bash
bash scripts/example.sh
```
